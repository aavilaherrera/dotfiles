# Dotfiles

## Quick install

```bash
git clone https://github.com/aavilaherrera/dotfiles.git dotfiles
cd dotfiles
./install default                   # basic defaults
# ./install default osx-admin brew  # install brew
# ./install brew-formulae           # brew some basic formulae
```
